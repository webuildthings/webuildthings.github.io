c程序内存布局

发表于 2020-05-22

一个典型的可执行的C语言程序在内存中应该有以下几部份组成：

1.代码段 (.TXT)

代码段中存放可执行的指令，在内存中，为了保证不会因为堆栈溢出被覆盖，将其放在了堆栈段下面

2. 初始化数据段（.DATA）

初始化数据段有时就称之为数据段.

3. 未初始化数据段（.BSS）

未显示初始化的全局变量和静态变量.

4. 栈

存放函数的局部变量,参数和返回地址

5. 堆

存放动态分配的空间.必须手动申请，手动释放,否则会出现内存泄漏。

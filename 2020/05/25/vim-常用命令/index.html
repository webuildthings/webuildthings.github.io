vim 常用命令

发表于 2020-05-25 更新于 2020-09-01

vi是大多数linux内建的文本编辑器。vi的学习成本相对比较高，但学会后工作效率将会大大提高。

定位

:　　　　后边加行号就是跳到这一行
f　　　　后后搜索，并将光标停留在目标字符上。（当前行有效)
F　　　　后前搜索，并将光标停留在目标字符上。（当前行有效)
t　　　　向前搜索并将光标停留在目标字符的前一个字符上.（当前行有效)
T　　　　后搜索并将光标停留在目标字符的后一个字符上.（当前行有效)

“``“　　　　到上次光标处(两个`符号)
`.　　　　　　命令相当强大, 它移动光标到上一次的修改点

m(a-z)　　　　书签, 用一个字母来标记当前位置，如用mz表示标记z
‘(a-z)　　　　跳转到指定的书签，如用’z表示移动到z书签

%　　　　跳到匹配的括号处（”{ }””[]””()”）

[[　　　　向前跳到顶格的第一个”{“
[]　　　　向前跳到顶格的第一个”}”
]]　　　　向后跳到顶格的第一个”{“
][　　　　向后跳到顶格的第一个”}”

[{　　　　跳到本代码块(由{}界定)的开头 , =%
[}　　　　跳到本代码块的结尾, =%

删除

dd　　　　剪切当前行
dib　　　剪切内含 ‘(‘ ‘)’ 块
dab　　　剪切一个 ‘(‘ ‘)’ 块
diB　　　剪切内含 ‘{ ‘ ‘ }’ 大块
daB　　　剪切一个 ‘{ ‘ ‘ }’ 大块
D　　　　剪切当前字符至行尾，D=d$
